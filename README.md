# vi-cheatsheet

### searching
- to look for text
- next match:  *n* 
- prev match: *Shift+n*
```
/<textToBeFound>
```
### goto
- end of file: *Shift+g*
- beginning of file: *gg*
- beginning of line: *0*
- end of line: *$*
- line no
```
:<lineNumber>
```
- one word right *w*
- one word left *b*
### select
- select line: *V*
- select text: *v*
- copy selection: *y*
- paste selection: *p*
- delete selection: *d*
- duplicate line *yyp*

### edit
- undo: *u*
- rendo: *Ctrl+r*

### various
- autocomplete: *Ctrl+p*
