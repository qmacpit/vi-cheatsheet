# vi-cheatsheet

### searching
- to look for text
- next match:  *n* 
- prev match: *Shift+n*
```
/<textToBeFound>
```

### goto
- end of file: *Shift+g*
- beginning of file: *gg*
- beginning of line: *0*
- end of line: *$*
- line no
```
:<lineNumber>
```

### select
- select line: *V*
- select text: *v*
- copy selection: *y*
- paste selection: *y*
- delete selection: *d*
